# NOOMA Click & Collect 🥐☕

Page Click & Collect pour NOOMA Babkery avec intégration Shopify.

## 🚀 Démarrage rapide

### Installation

```bash
pnpm install
```

### Configuration

1. Créez un fichier `.env` à la racine :

```env
VITE_SHOPIFY_DOMAIN=votre-boutique.myshopify.com
VITE_SHOPIFY_STOREFRONT_ACCESS_TOKEN=votre_token_ici
```

2. Lancez le serveur de développement :

```bash
pnpm dev
```

3. Ouvrez [http://localhost:5173](http://localhost:5173)
